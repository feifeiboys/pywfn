## 各个函数的作用
### script.py

#### `function(centerPos,aroundPos,alphas,cs,ts)`
获取在某轨道下中心原子周围某处的函数值
`centerPos`：中心原子坐标
`aroundPos`：周围某一点坐标
#### `get_functionValue(center,aroundPos,obtial)`
对function的封装，获取在某轨道下中心原子周围某处的函数值
`center`：中心原子序号
`aroundPos`：周围某一点坐标
`obtial`：轨道序号

## 变量命名原则

- center 代表中心原子
- around 代表与之相连原子

## 键级计算程序使用流程
1. 运行程序
2. 读取文件
    2.1 点击open
    2.2 在文件选择对话框中选中log或out文件
    2.3 等待读取完成
3. 指定原子
    3.1 在输入框中输入需要计算键级的原子序号
    3.2 点击自动获取按钮，获取与指定原子连接的原子
    3.3 点击计算轨道和键级按钮开始计算
4. 查看并保存计算结果
    3.1 在主页面中可以查看计算结果
    3.2 点击save按钮可以将计算结果以文档的形式保存

三个原子，找到这三个原子确定的平面看周围原子
    判断周围原子连接几个原子
    两个原子，则这个原子的法向量与中心原子相同

两个原子，说明是线型的轨道
