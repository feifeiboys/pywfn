## 各个函数的作用
### script.py

#### `function(centerPos,aroundPos,alphas,cs,ts)`
获取在某轨道下中心原子周围某处的函数值
`centerPos`：中心原子坐标
`aroundPos`：周围某一点坐标
#### `get_functionValue(center,aroundPos,obtial)`
对function的封装，获取在某轨道下中心原子周围某处的函数值
`center`：中心原子序号
`aroundPos`：周围某一点坐标
`obtial`：轨道序号

# Π轨道挑选方法
## p轨道贡献比例
1. 计算原子2PX,2PY,2PZ,3PX,3PY,3P归一化后的贡献P
2. 正常情况下，Π轨道P的贡献不能太小，否则为s轨道占主要贡献
3. 所以，当P<某一数值时不是Π轨道

## 键轴上函数值(已作废)
1. 计算两原子键轴中心处函数值V
2. 若函数值太大，则不是派轨道

## 键轴上10点函数值变化(已作废)
1. 在两原子间的键轴上平均取10个点
2. 将这10个点分别带入两原子的函数得到两组函数值
3. 将两组函数值相加取平均值，计算其标准差
4. 若标准差>某一数值则不是Π轨道

## 法向量
1. 获取与指定原子相连的三个原子坐标
2. 根据这三个原子坐标可以确定一个平面以及其法向量
3. 计算中心原子与相邻原子法向量(cn,an)和法向量反方向，距离原子0.2处函数值(cv1,cv2,av1,av2)
4. 理论上，Π轨道在垂直平面(即法向量)方向上下函数值应为一正一负
5. 所以，当cv1 * cv2<0 并且 av1 * av2<0 时才是Π轨道

## 垂直键轴与法向量
1. 计算法向量上距离原子0.2处的函数值cv
2. 计算垂直于键轴和法向量的向量，取距离0.2，计算函数值cv1
3. 进行比较，按正常情况下法向量上的函数值应该大于该向量,即abs(cv)>abs(cv1)
4. 所以，如果abs(cv)-abs(cv1)<0,则不是Π轨道

## 法向量与p轨道夹角
1. 计算原子周围空间格点处函数值，范围为-1，1，间隔为0.1，所以共8000个点
2. 找到最大和最小函数值所在的坐标，计算该坐标-原子坐标的向量p
3. 计算法向量n与p的夹角angle，理论上angle不应该太大
4. 所以，如果angle>某一数值则不是Π轨道

## 区别目标原子类型
计算目标原子连接原子的数量N
N==3时，计算法向量，所有分子轨道法向量相同
N==2时，计算p轨道法相，不同分子轨道的p轨道法相不同
