# log/out文件管理器
- 模仿zetero，这个就是一个大列表
- 设置本地存储文件夹，每一个文件都单独存放在一个随机命名的文件夹中
- 默认情况下文件会被拷贝到存储路径中，当然也可以保存原始路径信息而不复制文件
- 每一个文件夹内应该有个info.yml文件，用来保存文件的信息
- 用户可以直接通过该程序打开log文件，并且修改后程序会更新文件的信息
- 可以为文件添加 标签、分类、备注
    - 文件的类别用`route`表示，类似于路径:`a\b\c`
- 点击一个文件的时候，右侧预览窗口可以显示详细信息
ui界面是使用qt还是tk呢？之前使用qt是因为要和pyvista联用

程序界面分为三个大部分
1. Tree
用来显示文件分类
2. Table
用来显示该分类下的文件
3. Prop
用来预览文件的信息(图片+文本)
