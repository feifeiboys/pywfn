程序主要功能

菜单栏按钮 功能
open 打开文件
view->web 在浏览器中显示分子信息
view->default 调用系统默认方式打开读取文件
tools->计算键级 计算指定原子的键级，自由价以及电荷密度
tools->挑选轨道 输入指定原子和轨道序号，将结果导出至excel文件中
tools->渲染云图 在浏览器中显示p轨道点云图
save 将当前窗口内的信息保存为文本文件
config 修改程序的一些配置信息
help 展示程序使用说明书

自由价计算流程

1. 首先在项目路径中找到start.bat，双击运行,如果运行失败，可以在根目录打开命令行，输入python39/python.exe main.py

2. 程序运行后，点击菜单栏上的open,在弹出的文件选择对话框中选择需要处理的log/out文件

3. 文件读取完成后，点击tools->计算键级，弹出计算键级自由价子页面

4. 在子页面中首先输入需要计算的原子序号，然后点击自动获取与目标原子相连的原子，再点击计算键级即可

5. 在主窗口查看计算结果，包含每个原子与邻接原子的Π轨道，键级以及原子自由价与电荷密度

警告

1. 在程序中新打开文件时，要重新开启子页面，每个子页面只能计算一个log文件
